## Requirements

You need to have

- A postgresql database
- A redis database
- Nodejs (and also Yarn)

## Running

- Run `yarn` inside the `./client` and `./server` folder
- Provide the `.env` inside `./server` with the necessary information:
  - `DATABASE_URL`: the url to the postgres database
- ...
